# CRDT Examples
Examples of CRDTs implementations. (For educational purpose only).


# State CvRDT
    State CvRDT is defined by (S s0 q u m)
        S   -> Global State
        s0  -> State at beginning
        q   -> Query method
        u   -> Update method
        m   -> Merge method
